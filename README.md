# Portafolio de prácticas preprofesionales

Portafolio digital de **Chabely Dianeth Parizaca Pinaso**, desarrollado a partir de las actividades realizadas en el **Colegio de Contadores Públicos de Puno**.

## Datos principales

- Practicante: Chabely Dianeth Parizaca Pinaso
- Centro de prácticas: Colegio de Contadores Públicos de Puno
- Área: Comunicación Organizacional
- Periodo: 13 de abril al 13 de julio de 2026
- Actividades: 23
- Ejes de trabajo: 8

## Ver el portafolio

La versión publicada estará disponible en:

**https://chabelypinazo-eng.github.io/portafolio-practicas-ccpp-puno/**

La navegación funciona dentro de una sola aplicación. Desde el inicio se puede ingresar a los ejes, consultar las 23 actividades y abrir el detalle de cada una sin cargar páginas independientes.

Cada actividad incluye estadísticas de sus metas y un visor integrado para recorrer fotografías, videos y documentos con flechas. Los cinco archivos PDF se muestran como libros digitales con navegación entre páginas.

Para abrirlo de forma local en Windows, utiliza el archivo `ABRIR PORTAFOLIO.bat` o abre `index.html`.

## Contenido

El repositorio incluye la interfaz web, las fotografías, cinco videos optimizados para reproducción web, cinco archivos PDF y los demás materiales vinculados a cada actividad. Las evidencias se conservan dentro de las carpetas `Actividad 1` a `Actividad 23`.

## Ejes de trabajo

1. Comunicación interna
2. Relaciones humanas
3. Comunicación externa
4. Protocolo y organización de eventos
5. Prensa institucional
6. Identidad institucional
7. Producción audiovisual
8. Web y redes sociales

## Archivos principales

- `index.html`: entrada única del portafolio.
- `styles.css`: diseño adaptable para escritorio y celular.
- `script.js`: navegación interna, filtros, estadísticas y visor multimedia.
- `data.js`: contenido de los ejes, actividades y evidencias.
- `ABRIR PORTAFOLIO.bat`: acceso directo para abrir la versión local.
